<h1>Gerador de senhas desenvolvido em HTML, CSS e JavaScript</h1>

<p>  Utilizei HTML, CSS e JavaScript para criar uma interface de usuário simples e fácil de usar que permite aos usuários gerar senhas aleatórias com diferentes níveis de complexidade e segurança. A senha gerada é exibida em tempo real na página, permitindo que o usuário copie e cole a senha onde for necessário.</p> 
<p>Para criar senhas aleatórias, utilizei algoritmos de geração de números aleatórios em JavaScript, combinando diferentes caracteres e símbolos para criar senhas seguras e complexas.</p> 
<p>Essa experiência me permitiu aprimorar minhas habilidades em JavaScript, além de aprender novas técnicas de criptografia e segurança de dados.</p>
